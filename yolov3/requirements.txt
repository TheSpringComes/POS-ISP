In this paper, we propose POS-ISP, a task-aware ISP optimization framework that achieves robust performance across diverse downstream tasks.
Previous approaches jointly optimize both module sequence and parameters via neural architecture search (NAS) or reinforcement learning (RL).
NAS-based methods employ mixture training, blending multiple modules during training but selecting only one at inference, which introduces a mismatch that degrades performance. RL-based methods rely on step-wise decision-making, suffering from unstable future reward estimation and high computational cost. 
In contrast, POS-ISP adopts a sequence-level RL formulation that eliminates the step-wise supervision. It predicts both the module sequence and all parameters in a single forward pass, updating exclusively from the final output.

The framework consists of sequence predictor, which determines the module order, and parameter predictor, which predicts the corresponding parameters based on the image. 
This design provides optimization stability and effectively captures global module interactions, enabling robust adaptation to various tasks with strong performance and significantly lower computational and memory overhead.
Extensive experiments across various downstream tasks show that POS-ISP achieves state-of-the-art performance with substantially lower resource overhead.
